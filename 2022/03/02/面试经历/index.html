Start和Awake有什么区别

  • Awake ：始终在任何 Start 函数之前并在实例化预制件之后调用此函数。（如果游戏对象在启动期间处于非活动状态，则在激活之后才会调用 Awake。）
  • Start ：仅当启用脚本实例后，才会在第一次帧更新之前调用 Start。
